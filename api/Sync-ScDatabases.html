Sync-ScDatabases

Sync-ScDatabases

Deserialize all items to the database.

Syntax

Sync-ScDatabases [[-ProjectPath] <String>] [<CommonParameters>]

Description

Deserialize all items from a repository to the database. This includes:

  • Perform unicorn sync for application and test data
  • Perform an update database for the default items

Parameters

-ProjectPath <String>

The path to the project for which the items should be installed.

Position Required Default value Accept pipeline input Accept wildchard characters
1 false false false
